# electron-js-hello-world

A simple TypeScript CLI that says hello :wave:.

## Usage

After installing dependencies and building, you can run:

```
npm start
```

or directly call the CLI (optionally passing a name):

```
npx hello-world-cli Alice
```

which prints:

```
Hello, Alice!
```

Without a name:

```
npx hello-world-cli
```

prints:

```
Hello!
```

## Development

- Build: `npm run build`
- Test: `npm run test`

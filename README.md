# electron-js-hello-world

A simple TypeScript CLI that says hello, with tests.

## Usage

After installing dependencies and building, you can run:

```
npx hello-cli           # Outputs 'Hello, world!'
npx hello-cli Alice     # Outputs 'Hello, Alice!'
```

## Development

Build the TypeScript:

```
npm run build
```

Run the tests:

```
npm test
```
